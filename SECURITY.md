# Security Policy

## Supported Versions

This project is not publicly maintained as an open source product.
Security updates are not guaranteed for any public or shared copy unless explicitly stated by the copyright holders.

| Version | Supported |
| ------- | --------- |
| Current private version | ✅ |
| Older versions | ❌ |

## Reporting a Vulnerability

If you believe you have found a security vulnerability in this project, please report it privately to the project owners.

Do not disclose the vulnerability publicly before receiving a response.

Please include:
- a clear description of the issue
- steps to reproduce it
- affected files, modules, or features
- proof of concept, screenshots, or logs if available
- the potential impact
